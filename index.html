install

One command. Run it on your Linux machine.

bash <(curl -fsSL https://raw.githubusercontent.com/small-tools-collective/scripts/main/install.sh)

Requires: Linux, X11, bash, systemd. Takes about 2 minutes.

what is this

Two tools. One for everyday threat, one for emergencies.

version a — soft clean

Ctrl+Alt+S — closes messengers, wipes browser history, clears clipboard, locks screen.

Safe to use any time. Nothing is permanently deleted.

version b — panic wipe

Insert your panic USB → system arms. Press Ctrl+Alt+W → wipe triggers.

Permanently destroys sensitive files. Screen shows a kernel panic to anyone watching.

your panic USB

For version B you need a dedicated USB drive. Any drive works — the system watches for its hardware ID.

  • Get a USB drive you'll carry with you
  • During install, plug it in when prompted — it's detected automatically
  • That USB is now your kill switch. Inserting it arms the system.
  • Removing it disarms — wipe only triggers if you also press Ctrl+Alt+W
Version B is irreversible. The install script runs in simulation mode by default — test before enabling real wipe.

install

Download the repo and run the installer from PowerShell.

powershell -ExecutionPolicy Bypass -File windows\install.ps1

Or: right-click windows\install.ps1 → Run with PowerShell

/ / / / / / / / / / / / / / / / / / / / / / / / / / / / / / / / / / /

requirements

  • Windows 10 or 11
  • PowerShell 5.1 or later (included with Windows 10+)
  • AutoHotkey v2 — for keyboard shortcuts
    install with: winget install AutoHotkey.AutoHotkey

The installer checks for these and guides you if anything is missing.
